## Tecnologias

O portal utiliza Jekyll, HTML5 semântico, CSS moderno, JavaScript puro, Markdown, YAML e Liquid. A publicação foi preparada para GitHub Pages por meio do fluxo oficial de Pages e pode ser adaptada para compilação nativa quando as versões forem compatíveis.

## Tipografia

As pilhas tipográficas priorizam **Libre Baskerville** para títulos e **Inter** para interface, mas o projeto não distribui arquivos de fontes. Enquanto fontes locais não forem adicionadas, o navegador utiliza alternativas de sistema documentadas em `assets/fonts/README.md`.

## Ícones

Os pequenos ícones de interface presentes no código são formas vetoriais simples criadas para este modelo. Caso sejam substituídos por uma biblioteca, informe aqui o nome, a licença e o endereço oficial do projeto utilizado.

## Imagens

Todas as imagens incluídas inicialmente são placeholders locais e substituíveis. Registre nesta página o autor, a licença, a origem e as alterações realizadas em cada imagem real adicionada ao portal.

## Colaboradores

Inclua os nomes somente com autorização das pessoas envolvidas e descreva, quando apropriado, a contribuição editorial, pastoral, técnica ou artística de cada uma.
